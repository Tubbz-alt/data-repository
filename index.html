ENRAM data repository for vertical profiles of birds

This repository contains vertical profiles of birds (VP) generated by vol2bird from volume scans of over 100 weather radars in Europe. More information on how to browse, download, and use these data can be found on GitHub. Data in this repository are available under a Creative Commons Zero waiver.
