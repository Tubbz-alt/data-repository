ENRAM data repository for bird profiles detected by weather radars

On the BALTRAD infrastructure, a vol2bird pipeline generates vertical profiles of birds from weather radar volume scans for over 100 radars. To archive these data, the European Network for the Radar surveillance of Animal Movement (ENRAM) has set up this open data repository. Instructions on how to use it can be found here.

Browse to a year directory to see file coverage.
